| T-04-02 | Tampering (SQLi) | GET /api/gastos?q= | mitigate | `Gasto.concepto.ilike(f"%{q}%")` uses a SQLAlchemy bind parameter — no string interpolation into raw SQL. `from`/`to` typed as `date` (FastAPI validates) |
| T-04-03 | Elevation of Privilege | CORSMiddleware | mitigate | Explicit `allow_origins=["http://localhost:5173"]`, `allow_credentials=False`, `allow_methods=["GET"]`. Added before routers (outermost). Verified by `test_cors_header` |

| T-04-05 | Input Validation | path/query params | mitigate | `id: uuid.UUID` and `from_/to: date | None` are FastAPI/Pydantic-validated; malformed input → 422 before handler runs |
