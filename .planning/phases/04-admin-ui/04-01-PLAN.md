<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → FastAPI admin router | Untrusted HTTP requests; query params, path params, and JSON bodies all arrive unsanitized from the browser |
| FastAPI → local filesystem | `/images/{filename}` resolves to a local path; must not escape storage_path |
| FastAPI → Postgres (SQLAlchemy) | Filter and search params flow into SQLAlchemy queries; must use parameterized binds |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-4-01 | Tampering | GET /images/{filename} | mitigate | `Path(pattern=r"^[^/\\]+$")` rejects path separators at FastAPI validation layer; secondary: resolve both paths and verify `str(file_path).startswith(str(storage_root))` before serving (ASVS V5.3) |
| T-4-02 | Tampering | GET /invoices query params (q, proveedor, status, fecha_*) | mitigate | SQLAlchemy `.where()` with bound parameters — never string-concatenated SQL; `func.lower().like(f"%{value}%")` uses a parameterized bind, not interpolation (ASVS V5.3) |
| T-4-03 | Information Disclosure | All admin endpoints — no auth | accept | v1 demo deploy on localhost; UI-07 auth deferred by locked user decision; documented in admin.py module docstring |
| T-4-04 | Denial of Service | PATCH /invoices/{id} and PATCH /invoices/{id}/items/{item_id} — request body | accept | FastAPI default 1MB body limit active; Pydantic schema rejects unknown fields via model_config; no additional mitigation needed at this scale |
</threat_model>
